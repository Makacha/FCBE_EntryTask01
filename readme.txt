# FCBE_EntryTask01

Khởi động server (Cần đã cài đặt Node.js)
****************************
Mở terminal hoặc cmd tại thư mục project và nhập lệnh: npm install
Tiếp tục nhập lệnh: npm start
Ngay sau đó url của server sẽ hiện lên

Trang defaut
****************************
Gồm 3 chức năng:
    1. Get timestamp: Hiển thị timestamp tại tab mới
    2. Save log: Lưu lại log mới với thông tin đã nhập
    3. Get n latest log: Hiển thị n log gần nhất tại tab mới

Trang /timestamp
****************************
Trả về timestamp hiện tại

Trang /log?limit=n
****************************
Trả vể n log gần nhất
